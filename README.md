# Оценка надежности пароля

Этот мой учебный проект, который представляет собой простой скрипт на Python для оценки наджености пароля с использование библиотеки urwid. Он анализирует введенный по символьно пароль, выводя итоговый рейтинг в терминал от минимальной (от 0) до максимальной (12) надежности по параметрам: буквы нижнего или верхнего регистра, цифры, символов # или %, итоговая длина пароля.

## Установка

1. Убедитесь, что у вас установлен Python 3.
2. Установите необходимые зависимости с помощью pip:
```bash
pip install urwid
```
## Использование
1. Скачайте следующие файлы:
    - rating_password.py - главный файл;
    - Для остановки скрипта использовать кнопку 'Esc';
2. Запустите скрипт:
```bash
python ваш_скрипт.py
```
